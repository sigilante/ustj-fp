Machine epsilon.

The machine epsilon, or smallest value discernable from 1.0, is determined by the precision of the floating-point representation. The machine epsilon for a particular bit width is determined by setting two to the negative power of the number of bits used for the magnitude of the mantissa and accounting for the leading implicit bit 1; for 32-bit single-precision float this is #math42##tex2html_wrap_inline390#. Differences from 1.0 smaller than this cannot be represented in this bit width.
