27The problem of exactness is not exactly the same as reproducibility, but it is related. Exactness means that the result is the same as if the calculation were carried out to inﬁnite precision and then rounded to the appropriate number of bits. Reproducibility means that the result is the same across diﬀerent platforms. Reproducibitiy requires identical outcomes from the same inputs, while exactness requires the correct answer regardless of algorithmic path.
